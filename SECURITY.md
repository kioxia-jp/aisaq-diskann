# Security

If you find any vulnerabilities in this software, please report them using the following methods.

## Reporting Security Issues

**Please do not report security vulnerabilities through public GitHub issues.**

1. For vulnerabilities related to the DiskANN:
    - [Reporting vulnerabilities in DiskANN](https://github.com/microsoft/DiskANN/blob/main/SECURITY.md)

2. For vulnerabilities related to modifications made by Kioxia:
    - Please contact us at the following email address: kioxia-psirt@kioxia.com
